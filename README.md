# Functional Group Detector

A beginner-friendly cheminformatics project built using RDKit.

## Features

- Detect common functional groups
- Display molecular formula
- Calculate molecular weight
- Display complete molecular information

## Functional Groups

- Alcohol
- Aldehyde
- Ketone
- Carboxylic Acid
- Ester
- Ether
- Amine
- Amide
- Benzene Ring

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
